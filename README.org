*rfcview* is an Emacs tool for browsing, downloading, and reading IETF RFC documents inside Emacs. When you run *rfcview*, it updates the RFC index if necessary, then presents an interactive index buffer.

* Features

- Browse the full up-to-date IETF RFC index with fancy and informative look.
- Filter by favorites, recently read, or keyword search (scored and ranked)
- Automatically download and display RFCs on demand.
- Multi-format support driven by the rfc-index: TXT and PDF render inside Emacs (PDF requires pdf-tools); HTML and XML open in the browser via =browse-url=
- Rich read mode:
  - Applying style to RFC document
  - Jump by section
  - Clickable TOC entries, RFC cross-references, and URLs
  - History back / forward across followed links
  - Seamless page breaks; looks like one continuous page


* Installation

** use-package :vc (Emacs 30+)

#+BEGIN_SRC emacs-lisp
(use-package rfcview
  :vc (:url "https://github.com/zeph1e/rfcview.el" :rev :newest)
  :commands rfcview)
#+END_SRC

** use-package with straight.el

#+BEGIN_SRC emacs-lisp
(use-package rfcview
  :straight (:host github :repo "zeph1e/rfcview.el" :files ("*.el"))
  :commands rfcview)
#+END_SRC

** Manual

Place all four =.el= files (=rfcview.el=, =rfcview-core.el=, =rfcview-index.el=, =rfcview-reader.el=) in the same directory and add:

#+BEGIN_SRC emacs-lisp
;; e.g. if placed in ~/.emacs.d/rfcview/
(add-to-list 'load-path "~/.emacs.d/rfcview")
(autoload 'rfcview "rfcview" t t)
#+END_SRC

* Usage

=M-x rfcview= opens the =*RFC INDEX*= buffer.

| Key             | Action                                                             |
|-----------------+--------------------------------------------------------------------|
| =RET= / =SPC=   | Open RFC under point                                               |
| =n= / =p=       | Next / previous entry                                              |
| =TAB= / =S-TAB= | Next / previous button                                             |
| =v=             | Toggle favorite                                                    |
| =A=             | Show all                                                           |
| =F=             | Show favorites                                                     |
| =R=             | Show recently read                                                 |
| =K=             | Keyword search                                                     |
| =#=             | Jump to RFC by number                                              |
| =f=             | Jump to filter line                                                |
| =s=             | Toggle sort order (asc / desc by RFC number; All / Favorites only) |
| =g=             | Refresh index                                                      |
| =?=             | Help                                                               |
| =q=             | Bury buffer                                                        |

In an RFC read buffer:

| Key                   | Action                                                          |
|-----------------------+-----------------------------------------------------------------|
| =]= / =[=             | Next / previous section heading                                 |
| =j=                   | Jump to section by number or title                              |
| =TAB= / =S-TAB=       | Next / previous link (RFC ref, TOC entry, URL)                  |
| =RET=                 | Follow link under point                                         |
| =B= / =F=             | History back / forward (after following an RFC or section link) |
| =C-c C-b= / =C-c C-f= | Same, chord form                                                |
| =n= / =p=             | Next / previous line                                            |
| =b= / =f=             | Backward / forward char                                         |
| =+= / =-= / =0=       | Increase / decrease / reset font size                           |
| =o=                   | View raw cached file                                            |
| =?=                   | Help                                                            |
| =q=                   | Quit (refocus index if visible)                                 |

* Reading formats

rfcview reads the =(Format: ...)= trailer of each entry in the rfc-index and only tries formats the index advertises. The four supported values for =rfcview:preferred-format= are:

| Value   | Where it opens                                  | Cached locally? |
|---------+-------------------------------------------------+-----------------|
| =txt=   | Emacs read mode (default)                       | yes             |
| =pdf=   | Emacs =pdf-view-mode= (requires pdf-tools)      | yes             |
| =html=  | External browser via =browse-url=               | no              |
| =xml=   | External browser via =browse-url=               | no              |

** Selection rules

When the index lists =rfcview:preferred-format=, it leads; the remaining listed supported formats follow as fallback (so a broken =txt= link can still resolve to =pdf= if the index lists both). When the index does *not* list the preferred format, it is dropped — the index is authoritative — and the listed supported formats are tried instead. If nothing supported is listed, rfcview reports the RFC as unavailable.

Some RFCs only present as PDF (e.g. RFC 8); rfcview picks up the index's =(Format: PDF=...)= and opens the PDF directly. Likewise for HTML/XML-only entries.

** PDF support

Viewing PDFs requires [[https://github.com/vedang/pdf-tools][pdf-tools]]. rfcview does not declare a hard dependency on it; if pdf-tools is not installed, opening a PDF-only RFC signals an error. Example setup:

#+BEGIN_SRC emacs-lisp
(use-package pdf-tools
  :ensure t
  :magic ("%PDF" . pdf-view-mode)
  :config
  (pdf-tools-install :no-query))
#+END_SRC

** Customizing the preferred format

#+BEGIN_SRC emacs-lisp
  (use-package rfcview
    :vc (:url "https://github.com/zeph1e/rfcview.el" :rev :newest)
    :commands rfcview
    :custom (rfcview:preferred-format 'pdf))   ; or 'txt, 'html, 'xml
#+END_SRC
